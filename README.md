# findMyCar🚗
- **자동차 사진을 보고 어떤 기종인지 맞추는 자동차 차종 구 개인 토이 프로젝트**
- 일명 자차: (나의)자동차 찾기
  
## 1. 프로젝트 소개
1. 사용 데이터
- AIHub 제공 한국 이미지(차량) 데이터
- 누락 데이터 파악 후 수집 예정
2. 기술 스택
- 프론트: javascript, css
- 데이터: Python, pandas, jupyter, colab

3. 서비스에 대한 자세한 개요
- **자동차 찾기**: 자동차의 사진을 올리면 어떤 모델인지 찾아주는 서비스(**번호판은 가리기 필수**)
- **자동차 검색**: 내가 원하는 모델을 색깔과 브랜드, 크기 등을 통해 검색하는 서비스

## 2. 프로젝트 목표
1. 프로젝트 아이디어 동기
  
2. 프로젝트 메인 시스템 해결 방법 및 방향

3. 데이터를 통해 탐색하려는 문제

4. 서비스에 대한 자세한 개요

## 3. 프로젝트 기능 설명

2. 프로젝트만의 차별점, 기대 효과
