# findMyCar🚗
- **자동차 사진을 보고 어떤 기종인지 맞추는 자동차 차종 구별 개인 토이 프로젝트**
- 일명 자차: (나의)자동차 찾기
  
## 1. 프로젝트 소개
1. 사용 데이터
- AIHub 제공 한국 이미지(차량) 데이터(2019-12 갱신, 총 100종)
- 누락 데이터: 2020년 1월 이후 출시된 차종, 이후 보완 예정

2. 기술 스택
- 프론트엔드: javascript, css
- 데이터: Python, pandas, jupyternotebook

## 2. 프로젝트 목표
1. 프로젝트 아이디어 동기
- 가족과 함께 차로 이동할 때나 혼자 운전할 때나 도로 위에는 많은 차들이 있고, 나는 그 많은 차들의 이름을 모른다. 검색해보기에는 궁금한 차는 이미 지나가버리고 주변 사람에게 매번 물어보는 것도 눈치 보이게 된다. 그래서 내가 직접 찾을 수 있으면 하고 생각다가 떠올리게 됐다.

## 3. 프로젝트 기능 설명
- **자동차 찾기**: 자동차의 사진을 올리면 어떤 모델인지 찾아주는 서비스(**번호판은 가리기 필수**)
- **자동차 검색**: 내가 원하는 모델을 색깔과 브랜드, 크기 등을 통해 검색하는 서비스
- **자동차 정보**: 검색된 자동차에 대한 정보를 한눈에 제공하는 서비스(네이버 검색 기준)

## 4. 프로젝트 구성도
- 스토리보드: 파워포인트 또는 노션 사용 예정
- 와이어프레임: 피그마 사용 예정
